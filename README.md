## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
   ```
   git clone https://github.com/yasirSub/wesync-mcp.git
   ```

2. Navigate to the project directory
   ```
   cd wesync-mcp
   ```

3. Install dependencies
   ```
   flutter pub get
   ```

4. Run the app
   ```
   flutter run
   ```
